Built on the latest **Laravel 12** framework, it leverages **QR code technology** for seamless check-ins and provides a hierarchical role-based access control system to manage an organization's workflow efficiently.

## âš™ï¸ Installation

Follow these steps to set up the project locally:

1.  **Clone the Repository**

    ```bash
    git clone https://github.com/your-repo/c2c-attendance.git
    cd c2c-attendance
    ```

2.  **Install Dependencies**

    ```bash
    composer install
    npm install
    ```

3.  **Environment Setup**

    ```bash
    cp .env.example .env
    ```

    _Configure your database credentials in the `.env` file._

4.  **Generate Key & Migrate**

    ```bash
    php artisan key:generate
    php artisan migrate --seed
    ```

5.  **Build Assets**

    ```bash
    npm run build
    ```

6.  **Run the Server**
    ```bash
    php artisan serve
    ```
